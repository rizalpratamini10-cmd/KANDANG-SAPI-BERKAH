# Kandang Berkah Jaya | Balqys Aqiqah

## Sistem Website Qurban & Aqiqah

### Deskripsi
Website ini adalah sistem manajemen untuk peternakan Kambing dan Sapi yang menyediakan layanan Qurban, Aqiqah, dan Catering Service.

### Fitur Utama

#### Customer
- Login/Register menggunakan Nomor HP
- Melihat produk & jasa yang tersedia
- Pencarian produk (nama atau kode produk)
- Keranjang belanja dengan opsi DP atau Lunas
- Upload bukti transfer
- Melihat riwayat pesanan
- Invoice pesanan

#### Admin
- Dashboard dengan statistik
- Manajemen produk (stok 1, kode unik)
- Manajemen jasa
- Manajemen customer
- Manajemen pesanan (update status, follow up)
- Manajemen metode pembayaran (rekening bank)
- Laporan (penjualan, customer, keuangan, stok)
- Export laporan ke Excel & PDF

### Teknologi
- PHP Native
- MySQL Database
- HTML, CSS, JavaScript
- XAMPP / Laragon
